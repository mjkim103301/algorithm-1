# BOJ 2302번 [극장 좌석](https://www.acmicpc.net/problem/2302)

## 🌈 풀이 후기
* 순열 문제라고 생각했는데 dp여서 점화식을 고민하느라 오래 걸렸던 문제였습니다.

## 👩‍🏫 문제 풀이
* dp[0] =1
* dp[1]=1  
1
* dp[2]=2  
1, 2  
2, 1
* dp[3]=3
    * dp[2] 뒤에 3을 붙이는 경우   
    1,2,3  
    2,1,3
    * dp[1] 뒤에 2,3 을 순서 바꿔서 붙이는 경우  
    1,3,2
* dp[i]=dp[i-1]+dp[i-2]
